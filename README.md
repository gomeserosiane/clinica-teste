# Malcher USG Modern

Projeto institucional estático, moderno e minimalista para clínica de diagnóstico por imagem.

## Site institucional
Pasta: `site/`

Contém:
- Hero moderno com animações
- Sobre completo do médico
- Serviços com filtros por categoria
- Seção de depoimentos no corpo do site, sem link no header/footer
- Contato
- Botão flutuante do WhatsApp com mini container animado
- CSS separado por seção
- Arquivo exclusivo de responsividade
- JavaScript comentado

## Alterações aplicadas
- Blog removido do site.
- Links do Blog removidos do header e footer.
- CSS do Blog removido.
- JavaScript de posts, API, CRM e atualização em tempo real removido.
- CRM administrativo do Blog removido do projeto.
- Nova seção de Depoimentos criada no lugar do Blog.

## Como abrir localmente
```bash
cd site
# abra o index.html no navegador
```

## Deploy inicial na Vercel
O site está pronto como projeto estático. Basta publicar a pasta `site/` ou configurar o projeto para servir o `index.html`.
